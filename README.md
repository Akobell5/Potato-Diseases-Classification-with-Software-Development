**Potato Disease Classification using TensorFlow Lite and Cloud Integration**

This repository presents a **machine learning pipeline** for detecting and classifying **potato leaf diseases** using deep learning models optimized for deployment on **mobile and IoT platforms**.  
The project includes the **model training, TensorFlow Lite conversion, API backend, frontend web app, and mobile app integration** â€” providing an end-to-end smart agriculture solution.

**Overview**

Potato diseases significantly reduce crop yields, especially in developing regions.  
This project leverages **Convolutional Neural Networks (CNNs)** to automatically identify disease categories from leaf images and enables **real-time diagnosis** through edge or mobile deployment.
